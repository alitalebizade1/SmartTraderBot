# SmartTraderBot — پروژه جدید (نسخه آزمایشی)

این پوشه یک پروژه‌ی کاملاً مستقل و جدا از ریپازیتوری اصلی گیت‌هاب است.
هیچ فایلی از پروژه‌ی اصلی تغییر نکرده؛ بعد از رسیدن به نتیجه‌ی مطلوب،
می‌توانید این فایل‌ها را جایگزین نسخه‌ی روی گیت‌هاب کنید.

## استراتژی

الگوی «۳ برخورد به مقاومت / کف‌های صعودی»:

    روند صعودی قوی
        → برخورد ۱ به مقاومت → اصلاح → کف صعودی ۱ (HL1)
        → برخورد ۲ به مقاومت → اصلاح → کف صعودی ۲ (HL2 > HL1)
        → برخورد ۳ به مقاومت
        → سیگنال خرید (قبل از شکست مقاومت، با کندل تاییدی بالای HL2)

سطوح معامله:
- **Entry**: قیمت کندل تاییدی
- **Stop Loss**: دقیقاً روی HL2 (بدون بافر ATR)
- **TP1**: سطح مقاومت
- **TP2**: Measured Move = Entry + (مقاومت − شروع کانال)

## نصب و اجرا

نصب وابستگی‌ها از روی فایل requirements.txt، سپس اجرای main.py.

خروجی:
- گزارش کامل تحلیل در ترمینال (روند، ساختار الگو، سطوح معامله)
- فایل `chart_output.html` — چارت تعاملی که به‌طور خودکار در مرورگر باز می‌شود

## دیباگ (در صورت نبود سیگنال)

اجرای debug_run.py

این اسکریپت تمام Swing ها، انتقال‌های state machine، و دلایل رد هر
پترن را چاپ می‌کند تا بفهمید دقیقاً کجا الگو رد شده است.

## فایل‌ها

| فایل | توضیح |
|---|---|
| `main.py` | نقطه ورود اصلی — داده → تشخیص → گزارش → چارت |
| `config.py` | تنظیمات نماد/تایم‌فریم/بازه |
| `models.py` | مدل‌های داده (Swing, Pattern, Signal, ...) |
| `swing_detector.py` | تشخیص Swing با ATR تطبیقی |
| `pattern_detector.py` | State machine تشخیص الگوی ۳ برخورد |
| `pattern_validator.py` | اعتبارسنجی پترن قبل از صدور سیگنال |
| `indicators.py` | ATR/EMA/SMA و توابع کمکی |
| `chart.py` | رندر چارت تعاملی Plotly با نمایش کامل الگو |
| `report.py` | گزارش متنی فارسی از روند و سیگنال |
| `debug_run.py` | اجرای دیباگ با لاگ کامل هر مرحله |

## تنظیم نماد/تایم‌فریم

در `config.py`:

```
SYMBOL = "GC=F"     # طلا (نزدیک‌ترین معادل رایگان yfinance به XAU/USD)
INTERVAL = "15m"
PERIOD = "60d"      # محدودیت yfinance برای دیتای 15 دقیقه‌ای
```
